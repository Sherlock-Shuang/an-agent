# 零基础入门：从零手写你的第一个智能 Agent

这是一个为了“带你从原理理解 Agent”而打造的极简并且可用的本地代理(Agent)项目。
这个项目**没有**使用 LangChain 等封装得很重的框架，而是直接用最原生的 OpenAI 包为你展现“Tool Calling”核心逻辑。

## 功能介绍
它不只是个大语言模型聊天框。不仅能“说”，它还能“做”：
1. **网络搜索**：当它不知道时，它会自动调用 DuckDuckGo 搜索实时内容并汇总告诉你。
2. **文件读取**：它可以帮你读取你指定路径（如 `./agent.py`）的代码文件。
3. **文件写入**：当你跟它说“帮我写个贪吃蛇，保存到 `game.py`”，它真的会在当前文件夹创建这个文件并把代码写进去！

## 🚀 快速上手保姆级教程

### 1. 确保环境正常
请确认你的电脑上已经安装了 Python 3.8 以上版本。你可以在终端里输入：
```bash
python --version
```
> 如果提示命令未找到，你可能需要使用 `python3` 作为前缀，或者是去官网下载安装 Python。

### 2. 安装项目所需依赖包
在当前的 `an_agent` 目录下（或者说这个项目目录下），打开终端。运行：
```bash
pip install -r requirements.txt
```
> 这其中包含了 `openai` (调用模型的库)、`python-dotenv` (读取环境配置的库) 以及 `duckduckgo-search` (免费且不需要申请秘钥的搜索库)。

### 3. 配置你的模型秘钥 (API KEY)
你必须要有能够调用大模型的“通行证”（API 秘钥）。我们当前使用的是**阿里云百炼（通义千问）**的接口。通义千问提供了完全兼容 OpenAI 格式的调用接口，因此代码不需要大改！

1. 在当前文件夹下找到 `.env.example` 文件。
2. 把这个文件复制一份，并将其重命名为 `.env`（注意前面有个点）。
3. 用任何文本编辑器打开它，输入你在阿里云百炼申请的 API Key。

你的 `.env` 应该是这样的：
```env
OPENAI_API_KEY="sk-你的阿里云百炼秘钥"
OPENAI_BASE_URL="https://dashscope.aliyuncs.com/compatible-mode/v1"
MODEL_NAME="qwen-plus"
```

### 4. 运行开始玩耍
在项目目录下，执行：
```bash
python main.py
```
你就可以开始和它聊天，并且让它为你查资料，写文件了！
